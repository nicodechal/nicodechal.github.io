欧拉回路的判定

一个图上的 欧拉回路 指一条首尾相连的路径，该路径经过图上每条边，且只经过每条边一次。

一个图如果有欧拉回路，必需满足：

  • 每个顶点的度为偶数 ( 每个顶点的边有偶数条 )。

欧拉回路的求解

深度优先搜索 ( DFS )

确定一个图有欧拉回路后，取一个顶点开始进行 DFS，搜索时，要记录每条边的访问情况，去掉已经访问的边。

  • 从一个点出发，如果出现的回退，说明出现了回路。
